Ostrava: The Old Quarter

A level design exercise — six zones, one city, a spatial logic that tells its own story.

Overview

An academic level design presentation built around a fictional district of Ostrava, Czech Republic. The brief was to design a space with contextual logic — each zone had to feel like it belonged to the same city while serving a distinct gameplay function.

Design Approach
  • Six zones with distinct spatial rhythm, visual language, and gameplay purpose
  • Narrative context embedded in architecture — every area readable as a place with history
  • Custom Unity C# mechanics: climbing system and ledge detection via state machine
  • Verticality used as both gameplay tool and narrative metaphor
